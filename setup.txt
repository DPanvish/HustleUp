1) Install next js "npx create-next-app@latest ./"
2) Add extra properties to package.json for ensuring latest version of react is used across all the packages 
"packageManager": "npm@10.5.2",
"overrides": {
    "react": "$react",
    "react-dom": "$react-dom"
}
3) For Auth we are using Auth.js in Next.js "npm install next-auth@beta"
   -> For Auth Secret "npx auth secret"
   -> Complete the configuration by reading the docs in the Auth.js
   -> Add the providers
4) npm install tailwindcss-animate @tailwindcss/typography
5) npx shadcn@latest init
6) npx shadcn@latest add button for button in shadcn
7) It is recommended that componnts folder is placed outside app folder
8) Sanity Install "npm create sanity@latest -- --project r264oilc --dataset production --template clean"
9) To use sanity live api feature "npm install next-sanity@canary"
10) To markdown plugin in the schema "npm install sanity-plugin-markdown"
11) To extract the sanity schemas "npx sanity@latest schema extract --path=./sanity/extract.json"
12) Add the configuration file sanity-typegen.json
{
    "path": "./src/**/*.{ts,tsx,js,jsx}",
    "schema": "./sanity/extract.json",
    "generates": "./sanity/types.ts"
}

and run this command "npx sanity@latest typegen generate"
This will automatically create the type.ts for the schemas which contains 
the types of the schemas
13) Add the below configuration in the package.json scripts
"predev": "npm run typegen",
"prebuild": "npm run typegen",
"typegen": "sanity schema extract --path=./sanity/extract.json && sanity typegen generate"

Now we can run "npm run typegen" to generate types
14) Run this command "npm i server-only"
This will help us ensure that a specific module can be used in the server components
15) install "npm install markdown-it" and "npm i --save-dev @types/markdown-it"
16) install additional css components "npx shadcn@latest add input textarea toast"
17) For Pitch input field install react markdown editor "npm i @uiw/react-md-editor"
18) Create a validation.ts which uses zod (It is a TypeScript-first schema validation with static type inference)
19) Install slugify package "npm install slugify" which is used for geenerating unique id
20) Override the next in the package.json before setting up sentry, this will ensure that sentry uses
    the same version of next js that we are using in the project
    "overrides": {
        "react": "$react",
        "react-dom": "$react-dom",
        "next": "$next",
    },
20) Set up sentry follow the instructions on the site
21) Add Skeleton, Avatar component from shadcn 
    "npx shadcn@latest add skeleton avatar"
